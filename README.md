# Weekly Weather

Weekly Weather est une application réalisée en Flutter affichant les prévisions météo du jour, ainsi que ceux des sept jours suivants.

## Fonctionnalités

- Liste des jours de la semaine avec une icône de la météo, le jour et la température.
- Cliquer sur un jour de la semaine permet d'afficher une page plus détaillée (descritption, températures minimales et maximales, vent, précipitations, humidité).
- Les données sont mockées (écrites en dur).

## Lancer le projet

### Prérequis

- Installer [Flutter](https://docs.flutter.dev/get-started/install).
- Installer un émulateur Android tel que [Android Studio](https://developer.android.com/studio).

### Installation

```bash
git clone git@github.com:donata-dcz/weekly-weather.git
cd weekly_weather
flutter pub get
flutter run
```
